Spark Base-Kitchen

Sparktech Systems Pvt Ltd (SSPL) provides Spark Base Kitchen Software Solution for Central Kitchen Monitoring for single or multiple outlets.The software solution covers all the operations as part of central warehousing management . From Kitchen Production to Issues & Adjustment, it takes care of specific requirements of your centralized kitchen from where all the raw or semi-finished materials are being sent to the restaurant's kitchen or store.


Features of SSPL Base Kitchen Software:

STORES
  • Order Booking
  • Purchase Order Generation
  • Receiving
  • Issue to Production
  • Store Adjustment
KITCHEN
  • Production
  • Dispatch to own stores
  • Retail Invoice/ Tax Invoice
  • Separate Inventory for Raw/Finished Product
  • Calculation of Raw Item Requirement on basis of estimated production.

The following reports are generated from the system

  • Item Ledger
  • Semi Finished Production
  • Order Booking
  • Stock In Hand
  • Store Ledger
  • P.O. Report
  • M.R.N Report
  • Purchase Report
  • Booking Modify
  • Kitchen Wise Issue Report
  • Pending Report
  • Tax Register for Tax/Retail Invoices
  • Variance Report Qty Wise
  • Daily Purchase Report
  • Daily Consumption Report
  • Kitchen Costing Purchase
All the reports can be generated for a Day or for any period. Reports can generated on Screen, Printer or file. There is an option to export the reports to Excel, Word and send through email.
